#系统业务演示流程
##单条投诉处理流程
1.打开程序GUI界面
2.在输入框填写用户投诉内容,点击提交
3.程序调用分类函数自动识别投诉业务类型
4.根据分类调取对应售后回复展示在界面
5.系统自动将投诉内容、分类、回复三条数据写入data/result.csv报表
6.可打开Excel查看所有历史投诉工单

##批量文件处理流程
1.准备txt测试文件,每行一条用户投诉
2.在GUI点击导入文件按钮,选中本地测试文本
3.程序逐行读取每条投诉,循环执行分类、生成回复、保存CSV逻辑
4.批量处理完成后弹窗提示,直接打开CSV查看全部批量工单

##分类匹配示例 
输入：物流十天未送达 → 分类：物流问题
输入：商品收到外壳破损 → 分类：包装问题
输入：退款申请两周未到账 → 分类：退款问题
输入：产品出现开裂瑕疵 → 分类：产品质量问题
输入：咨询活动优惠券 → 分类：其他问题
