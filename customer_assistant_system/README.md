#Customer Assistant V3
##项目介绍
这是一款Python产现的企业客服自动化系统v3Demo

##技术栈
-Python
-TKinter(GUI界面)
-AI Prompt 工程
-自动化处理
-AI API 调用

##a项目结构
'''

##
使用愿意付出
1.环境要求:安装Python
2.终端进入项目根目录，运行:
'''bash
python app.py
'''
3.上传complaint.txt投诉文件
4.系统自动读取内容，生成回复，保存结果文件

##项目目标
练习Python工程化开发，AI自动化落地。桌面GUI开发，打造可展示的实战项目。
