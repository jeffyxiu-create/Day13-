# 文件自动归档助手 V1.5（终极优化企业版）
基于 Python + Tkinter 开发的桌面自动化文件整理工具，零基础可运行、企业级稳定，适合日常办公自动化、Python桌面开发作品集展示。

## 🚀 本次终极核心优化点
1. 多线程异步运行，**彻底解决GUI界面卡死、假死问题**
2. 采用 shutil.move 替代原生 rename，支持跨磁盘稳定移动文件
3. 全局异常捕获，单个文件处理失败不中断整体任务、不崩溃
4. 新增文件防覆盖机制，完全保护原始数据，零丢失风险
5. 重构查重逻辑：**全名+后缀精准匹配**，彻底杜绝同名不同格式文件误判重复
6. 线程安全UI刷新机制，无闪退、无后台报错

## ✨ 完整核心功能
- 智能分类：自动区分 PDF、Word、Excel、图片、其他文件
- 数据统计：精准统计各类文件整理数量
- 重复检测：全局扫描重复文件并列出完整路径
- 日志留存：自动生成带时间戳的每日整理日志
- 报表导出：一键导出 Excel 可打开的中文无乱码CSV报表
- 可视化GUI：极简桌面窗口，一键操作、无需命令行

## 💻 使用教程
1. 克隆/下载项目全部文件
2. 直接运行 `main.py`
3. 点击【选择整理文件夹】指定目标目录
4. 点击【开始自动整理】一键执行
5. 整理完成自动展示统计结果、重复文件、文件保存路径
