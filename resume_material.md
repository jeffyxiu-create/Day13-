# 项目整体说吸:客户投诉管理系统
## 项目定位
面向中小型电商企业的轻量化售后投诉处理工具.替代人工手动回复、手工记录工单.降低今后人力成本.

## 技术栈
Python 内置标准库:tkinter(GUI)、csv(报表)、os(文件路径管理)
无需额外第三方依赖包,开箱即可运行

##  核心业务流程
用户投诉输入/批量文件导入 → 文本关键词自动分类 → 匹配对应售后回复 → 数据自动写入csv业务报表 → 完成工单留存

## 项目模块拆分 
1.app.py: 程序统一入口,串联全部模块
2.ui.py:tkinter: 图形界面,提供可视化操作
3.file_loader.py: 本地文本文件读取功能
4.reply_engine.py: 投诉分类引擎、回复话术库
5.result_save.py: csv报表持久化存储
6.data文件夹: 存放测度用例、生成报表、话术文本
7.docs文件夹: 存放开发日志、项目说明、业务流程文档

## 项目成果
落地一夽轻量化售后自动化工具,替代人工逐条登记、回复工单,提升客服整理效率;项目模块化架构,代码上传GitHub附带演示视频.
