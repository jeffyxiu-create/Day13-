
# 013 OpenAI API & PDF 常见错误
## 错误1：401 Unauthorized
现象：API调用失败，返回401
原因：API Key错误、Base URL错误、模型名称错误
解决：核对配置信息，确认密钥有效

## 错误2：请求超时
现象：Request Timeout
原因：网络异常、接口响应慢
解决：增加代码内 timeout 参数，切换网络重试

## 错误3：内容过长 Token超限
现象：请求被拒绝，提示Token超出限制
原因：PDF原文体量过大
解决：代码内截取前5000字符，界面给出截断提示

## 错误4：图片型PDF读取为空
现象：PDF正常打开，但提取文本为空
原因：文件为扫描图片版PDF，无可复制文字
解决：提示用户当前为图片PDF，暂不支持解析

## 错误5：文件权限/路径错误
现象：读取PDF、保存文件失败
原因：文件被占用、路径含特殊字符、无读写权限
解决：关闭占用文件，更换保存路径
